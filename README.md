# DragonFly Lotus Command Center v2

## Install in GitHub Codespaces
1. Back up your current project.
2. Upload `index.html`, `styles.css`, and `app.js` into the repository root.
3. Replace the existing files when prompted.
4. Refresh the preview.
5. Test Health Dashboard values and confirm Captain's Briefing updates immediately.

## Commit
```bash
git add index.html styles.css app.js
git commit -m "Install DragonFly Lotus Command Center v2"
git push
```

## Data
This version continues using browser localStorage. Data stays on the device/browser where it was entered.
